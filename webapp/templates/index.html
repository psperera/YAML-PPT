Quick Start

  1. Click Load Template to start with a sample
  2. Edit the YAML content in the editor
  3. Click Save YAML to save your work
  4. Click Validate to check your YAML
  5. Click Generate PPT to create your presentation
  6. Download the generated PowerPoint file

Available Slide Types

  • title_white - White title slide
  • title_reverse - Reverse title slide
  • divider - Section divider
  • text_only - Text content slide
  • title_only - Title only (for charts)
  • two_column - Two column layout
  • three_column - Three column layout
  • quote - Quote slide
  • end_slide - Closing slide

Tips

  • Use | for multi-line text content
  • YAML is case-sensitive and indentation matters
  • Always validate before generating
  • Check the console for detailed error messages
